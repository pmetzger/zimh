## `3b2_400_maint_nvram.bin`

- Source: generated by this repository from `3b2_400.bin` and
  `3B2_Maintenance_Utilities_4-0.img`.
- Generation method: boot the 3B2/400 ROM with the Maintenance Utilities
  diskette attached, enter the firmware maintenance path, run `dgmon`,
  answer `n` to the `filledt` question, and stop after `filledt`
  completes and returns to the firmware program prompt.
- Generated: 2026-04-26
- Size: 4096 bytes
- SHA-256:
  `2f5f099ad9370b6ca6118e6a74adf490385bb18277ca3af309f775eb030e971c`
- Notes: NVRAM state for the 3B2/400 Maintenance Utilities smoke test.
  Tests must copy this file to a disposable output path before attaching
  it, because the NVRAM device does not support read-only attach.
